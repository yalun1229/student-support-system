# 留学生学習支援Webシステム

## 概要
このシステムは、留学生の学習記録や学習目標を管理するためのWebアプリケーションです。  
学習時間を記録し、グラフや達成率を通して学習状況を可視化できます。

## 開発目的
私は留学生として、日本での学習管理や生活情報の整理に難しさを感じました。  
その経験から、留学生が自分の学習状況を簡単に管理できるシステムを作成しました。

## 使用技術
- Python
- Flask
- SQLite
- HTML
- CSS
- Chart.js

## 主な機能
- ログイン機能
- 学習記録の追加・編集・削除
- 学習時間のグラフ表示
- 学習目標の登録
- 達成率の表示
- ナビゲーションバー
