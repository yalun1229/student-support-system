# 画像情報と生成AIを活用した留学生向け専門学習支援システム

## 概要

本システムは、日本で専門分野を学ぶ留学生を対象とした学習支援Webシステムです。

留学生が専門科目を学習する際には、日本語だけでなく、専門用語、授業資料、板書、プログラム画面などを理解する必要があります。特にIT分野では、専門用語や英語表記、プログラムコードが多く含まれるため、学習内容を整理したり、分からない点を日本語で質問したりすることが難しい場合があります。

本システムでは、留学生が学習内容を記録し、学習時間や学習目標を管理できる機能を実装しています。今後は、画像情報と生成AIを活用し、授業資料やノート画像から学習内容を整理・説明する機能を追加する予定です。

## 研究背景

近年、日本で専門分野を学ぶ留学生は増加しています。しかし、専門科目の学習では、日常会話の日本語能力だけでは十分でない場合があります。

特に、IT分野では以下のような困難があります。

- 専門用語の意味が分かりにくい
- 授業資料や板書の内容をすぐに理解できない
- プログラム画面やエラー内容を整理しにくい
- 分からない内容を日本語で質問することが難しい
- 学習内容を継続的に管理することが難しい

このような課題を解決するために、留学生の専門学習を支援するWebシステムを開発しました。

## 研究目的

本研究の目的は、画像情報と生成AIを活用し、留学生が専門学習の内容を理解しやすくするための学習支援システムを開発することです。

現在は、学習記録、学習時間の可視化、学習目標管理を中心とした原型システムを作成しています。今後は、画像アップロード機能、OCRによる文字抽出機能、生成AIによる説明生成機能を追加し、より実用的な学習支援システムへ発展させる予定です。

## 主な機能

現在実装している機能は以下の通りです。

- ログイン機能
- 学習内容の記録
- 学習時間の登録
- 学習記録の編集・削除
- 学習時間のグラフ表示
- 学習目標の設定
- 目標達成率の表示

## 今後追加予定の機能

今後は、研究計画に基づき、以下の機能を追加する予定です。

- 授業資料、板書、ノート画像のアップロード機能
- OCRによる画像内文字の抽出機能
- 生成AIによる専門用語の説明機能
- 学習内容の要約機能
- 教員へ質問するための日本語文作成支援機能
- 留学生へのアンケートや利用記録による有効性評価

## 実行方法

1. 必要なライブラリをインストールする

```bash
pip install flask
```

2. Flaskアプリを実行する

```bash
python app.py
```

3. ブラウザで以下のURLにアクセスする

```text
http://127.0.0.1:5000
```
